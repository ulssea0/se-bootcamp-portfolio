## 💬 Reflection

### สิ่งที่ฉันได้เรียนรู้มากที่สุด
- สิ่งที่ได้เรียนรู้มากที่สุดคือการคิดอย่างเป็นระบบก่อนลงมือทำงาน ไม่ว่าจะเป็นการวิเคราะห์ปัญหา การออกแบบ Flowchart หรือการเขียนโปรแกรม การวางแผนขั้นตอนการทำงานที่ชัดเจนช่วยให้สามารถแก้ปัญหาได้ง่ายขึ้น นอกจากนี้ยังได้เรียนรู้การใช้งาน Git และ GitHub ซึ่งเป็นเครื่องมือสำคัญในการจัดการและติดตามการเปลี่ยนแปลงของโปรเจกต์

### เรื่องที่ยังไม่เข้าใจ
- ปัจจุบันยังมีบางเรื่องที่ต้องศึกษาเพิ่มเติม เช่น การทำงานเบื้องหลังของระบบเครือข่าย การเชื่อมต่อระหว่าง Front-end และ Back-end รวมถึงการใช้งาน Git ในกรณีที่มีการทำงานร่วมกันหลายคน เช่น Branch, Merge และการแก้ไขปัญหาเมื่อเกิด Conflict

### สิ่งที่ฉันจะฝึกต่อ
- ตั้งใจจะฝึกเขียน Flowchart และการคิดเชิงตรรกะให้คล่องมากขึ้น รวมถึงศึกษาการใช้งาน Git และ GitHub เพิ่มเติม เพื่อให้สามารถใช้งานได้อย่างมั่นใจ นอกจากนี้ยังอยากฝึกพัฒนาเว็บไซต์และโปรเจกต์ขนาดเล็กด้วยตนเอง เพื่อเพิ่มประสบการณ์และนำความรู้ที่เรียนมาไปประยุกต์ใช้จริง
